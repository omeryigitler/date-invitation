# Date Invitation

Romantik ve etkileşimli bir date invitation uygulaması. React, Vite, Tailwind CSS, Motion ve date-fns ile hazırlanmıştır.

## Özellikler

- Taşma yapmayan responsive teklif ekranı
- Kaçan `Hayır` butonu
- Tarih ve gerçek saat seçimi
- Sabah / Öğle / Akşam öğün seçimi
- Yemek tercihi carousel'i
- Final onay ekranında tarih, saat, öğün ve menü özeti

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda şu adresi aç:

```bash
http://localhost:3000
```

## Build

```bash
npm run build
npm run preview
```
